技术栈选择

暗黑科技风仪表盘的技术底座

A

Next.js 14 + Tailwind CSS + shadcn/ui

App Router + Server Components + Tailwind 原子化样式 + shadcn/ui 组件库。暗黑模式开箱即用，shadcn/ui 的组件风格天然适合暗黑科技风。

🎨 暗黑适配：Tailwind dark: 前缀，完美控制
🧩 组件：shadcn/ui Card/Badge/Tabs 即插即用
⚡ 性能：RSC + Tailwind JIT，极致性能
B

Next.js 14 + CSS Modules + 手写组件

App Router + CSS Modules 手写所有样式和组件。完全定制化，但开发量大。

🎨 自由度：100% 定制，无框架约束
⚠️ 成本：所有组件从零写起，耗时长

其他技术选型

数据库 Supabase (PostgreSQL) — 已有 schema
ORM @supabase/supabase-js (官方SDK，无需Prisma)
内容格式 Markdown/MDX 存储，前端渲染
部署 Vercel (Next.js 原生) / 自部署
图标 Lucide Icons (与 shadcn/ui 配套)

💡 推荐 A（Tailwind + shadcn/ui） — 暗黑模式、组件质量、开发速度三者最优解。
